font-style is shorthand property can be used to italicize text.
At first glance, italic and oblique look similar.
If italic is a 'slanted' font, oblique is the original font that is slanted.

Each font may have both, one side only, or neither, They are used interchangeably when one side is missing.

font-weight is shorthand property can control the boldness of the text.

Depending on what weight your font supports, choose between normal and bold, or use a number between 100 and 900 in 100 increments.

font-size is shorthand property specifies the size of the text.

The units used are px, %, em, and rem.

px is an absolute value, expressed in pixels.
100% is 1em, and they represent the relative size to the parent element.
rem has a size relative to the html element, so it is not affected by element nesting.
pt is 1 inch/72 and is used for content to be printed.

text-decoration property is used to decorate text with
underlines, strikethroughs, wavy lines, etc.

This property can be used with multiple decorative elements, and
can also specify details such as line shape, color, and thickness.

text-transform property is used to display uppercase and lowercase letters of the alphabet.

Text written in uppercase letters, such as HTML and CSS, is not affected by capitalize.

text-shadow property can give text a shadow

You can insert a shadow in the format of 'x coordinate, y coordinate, blur (optional), color', and
You can also insert multiple ones separated by commas.
